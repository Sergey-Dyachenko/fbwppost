=== Fbwppost - Post your blog post to facebook page wall ===
Tags: Wordpress,facebook

== Installation ==

Installing Fbwppost plugin from your WordPress admin panel

1. Go to Plugins page, and add new plugin.
2. Search for Fbwppost and select fbwppost plugin from the list
3. Click Install Now  button
4. Then click Activate button
5. And use plugin!
